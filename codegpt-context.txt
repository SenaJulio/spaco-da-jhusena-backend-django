📘 CONTEXTO DO PROJETO — SPAÇO DA JHUSÉNA 🐶💚

O Spaço da Jhuséna é um sistema web completo desenvolvido em Django, criado para gerenciar um pet shop e oferecer um painel financeiro inteligente com IA integrada.

🎯 OBJETIVO
Automatizar o controle de receitas, despesas, agendamentos e gerar recomendações financeiras inteligentes com base em dados reais das transações do negócio.

---

🧩 ARQUITETURA PRINCIPAL

• Backend: Django (Python)
• Banco de dados: SQLite (padrão do Django)
• Frontend: HTML, Bootstrap 5, Chart.js e JavaScript (arquivos: dashboard.js, historico_ia.js)
• Painel de IA: Módulo que analisa receitas e despesas dos últimos 30 dias e gera dicas automáticas
• API endpoints: `/financeiro/ia/resumo-mensal/`, `/financeiro/dashboard/dados-filtrados/`, `/financeiro/api/insights/criar-simples/`
• Autenticação: sistema padrão do Django (login via /admin/)

---

📊 FUNCIONALIDADES

✅ Painel Financeiro Inteligente
- Exibe receitas, despesas e saldo atual
- Gera automaticamente dicas de IA baseadas no desempenho financeiro
- Histórico de recomendações da IA
- Gráfico de evolução financeira (Chart.js)
- Filtros por data e categoria
- Atualização via AJAX sem recarregar a página

✅ Modelos principais
- Transacao (tipo, valor, data, categoria)
- Insight (textos automáticos gerados pela IA)
- RecomendacaoIA (histórico de dicas personalizadas)

✅ Views importantes
- `dashboard_financeiro` → exibe o painel principal
- `ia_resumo_mensal` → retorna o resumo mensal via JSON
- `dados_grafico_filtrados` → envia dados filtrados para o gráfico
- `gerar_dica_30d` → gera dicas da IA com base nos últimos 30 dias

✅ Scripts front-end
- `dashboard.js` → renderiza gráficos e consome endpoints Django
- `historico_ia.js` → controla histórico e feed de recomendações

---

🤖 INTELIGÊNCIA ARTIFICIAL
O sistema contém uma mini-IA que:
- Calcula margem e desempenho do mês
- Gera insights automáticos ("Reforce a reserva", "Corte gastos", etc.)
- Armazena o histórico no modelo `RecomendacaoIA`
- Pode gerar dicas sob demanda via botão “Gerar Nova Dica com IA”
